# finalproject
To run Program: 
0. Open terminal
1. git clone https://github.com/SandeepReji/finalproject
2. cd APW_JS_MW       (may be differnt directory for you)
3. cd finalproject
4. cd Final2
5. npm run devStart   (script restarts the node application)
6. open browser
7. In the browser type:   localhost:3000
